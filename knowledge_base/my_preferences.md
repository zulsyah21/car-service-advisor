# Project Context & Preferences

**Last Updated:** 2026-06-14

## Project Overview
- **Name:** Web-Based Automated Car Service Advisor (Final Year Project - FYP)
- **Institution:** UiTM (Degree), Semester 5
- **Focus:** Automotive diagnostic advisor tool tailored for Perodua car models (e.g. Myvi, Axia, Bezza, Alza, Aruz, Ativa, Traz).
- **Core Stack:** Vanilla HTML/CSS/JS frontend, Node.js + Express backend, MySQL database, local storage seeding mechanism.

## Environment & Platform Preferences
- **OS:** Windows
- **Shell:** PowerShell (Avoid Linux/Bash-specific commands unless mapped to PowerShell syntax).
- **Paths:** Always use standard Windows-compatible formatting (forward slashes or double backslashes in JS scripts).

## Frontend Design Philosophy
- **Aesthetics:** High-end, premium dark-themed design system, smooth micro-animations, glassmorphism card designs, and vibrant gradients (e.g. HSL tailored color rules using crimson/red accents representing Perodua colors).
- **Frameworks:** Maintain vanilla CSS/JS structure to avoid Tailwind CSS/framework overhead unless requested.
- **Icons:** Font Awesome 6 standard elements (use standard icons e.g. `fa-wrench`, `fa-lightbulb`, `fa-car`, etc.).
